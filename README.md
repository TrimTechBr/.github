**TrimTechBR** is a collaborative GitHub organization created by **Henderson Andrade** and **Henrique Mizael** to share personal, open-source and community-driven technology projects.

The main goal of TrimTechBR is to build and publish practical solutions focused on cloud architecture, infrastructure automation, DevOps, Infrastructure as Code, observability, governance and modern IT operations.
This organization is a space for experimentation, collaboration and knowledge sharing, bringing together real-world experience from cloud and infrastructure architecture.

## Focus Areas

- Microsoft Azure
- Cloud Architecture
- Infrastructure as Code
- DevOps and CI/CD
- Terraform and Terragrunt
- PowerShell automation
- Bicep and ARM templates
- Kubernetes and GitOps
- Multi-cloud practices
- Microsoft 365 and cloud operations
- Observability and governance
