# TrimTechBR

🌐 Idioma: [English](README.md) | [Português do Brasil](README.pt-BR.md)

A **TrimTechBR** é uma organização colaborativa no GitHub criada por **Henderson Andrade** e **Henrique Mizael** para compartilhar projetos pessoais, open source e iniciativas voltadas à comunidade técnica.

O principal objetivo da TrimTechBR é construir e publicar soluções práticas com foco em arquitetura de nuvem, automação de infraestrutura, DevOps, Infrastructure as Code, observabilidade, governança e operações modernas de TI.

Esta organização é um espaço para experimentação, colaboração e compartilhamento de conhecimento, reunindo experiências reais em arquitetura de nuvem e infraestrutura.

## Áreas de Foco

- Microsoft Azure
- Arquitetura de Nuvem
- Infrastructure as Code
- DevOps e CI/CD
- Terraform e Terragrunt
- Automação com PowerShell
- Bicep e ARM Templates
- Kubernetes e GitOps
- Práticas multi-cloud
- Microsoft 365 e operações em nuvem
- Observabilidade e governança

## Projetos

Os repositórios desta organização normalmente são focados em:

- scripts de automação para Azure
- exemplos de Infrastructure as Code
- utilitários para governança em nuvem
- ferramentas de observabilidade e monitoração
